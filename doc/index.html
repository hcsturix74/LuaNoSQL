LuaNoSQL - Lua NoSQL database connectivity

Overview

LuaNoSQL is a simple interface from Lua to NoSQL DBMS (heavily inspired by LuaSQL). This implementation is NOT based on FFI library. It enables a Lua program to:

  • Connect to UnQLite, Vedis (not supported yet) databases;
  • Execute arbitrary key/value operations;
  • Manage data using cursors.

LuaNoSQL is free software and uses the same license as Lua 5.1.

Status

LuaNoSQL version 1.0.0 (for Lua 5.X) is now available for download. For more details on the features list please check the product Versions.

Download

Source code for LuaNoSQL can be downloaded from its GitHub repository.

Credits

This project has been heavily inspired by LuaSQL and LuaSQLite3 projects which helped me a lot in understanding how Lua/C bindings can be created. This implementation shares with LuaSQL the structure for managing: environment, connections and cursor objects.
